# Organizador-De-Arquivos
programa em python que organiza arquivos dentro de uma pasta, separando eles por suas extensões

Esse é um organizador de arquivos simples, que consiste em: selecionar uma pasta e executar o programa que irá criar novas pastas separando os arquivos por suas extensões

## Como usar

1. Clone o repositório ou baixe o arquivo `organizador.py`.
2. Certifique-se de ter o Python instalado em seu sistema.
3. Execute o programa usando o comando:
   ```bash
   python organizador.py
   ```
4. Selecione a pasta que deseja organizar.
5. O programa criará novas pastas dentro da pasta selecionada, organizando os arquivos por suas extensões.
